# AI Project Idea Generator

## Project Overview
An AI-powered tool that helps developers find meaningful, skill-aligned project ideas instead of generic tutorials.

## Problem it Solves
Engineering students often struggle to decide what to build or choose projects that are too large or too generic. This tool provides structured, actionable project directions.

## Tech Stack
* **Backend:** FastAPI (Python)
* **Frontend:** Streamlit
