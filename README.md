# AI Project Idea Generator

An AI-powered tool that helps developers find meaningful, skill-aligned project ideas instead of generic tutorials.

## Problem it Solves
Engineering students often struggle to decide what to build or choose projects that are too large or too generic. This tool provides structured, actionable project directions.

## Tech Stack
* **Backend:** FastAPI (Python)
* **Frontend:** Streamlit
* **LLM Integration:** OpenAI / Google Gemini

## Features
* **User Input Interface:** To collect skill level, preferred domain, and career goals.
* **AI Idea Generation Engine:** To generate 3–5 structured project ideas per request.
* **Structured Output:** Each idea includes project title, problem statement, pre-requisites, suggested features, tech stack hint, and difficulty level.

## Key Decisions
* Chose **FastAPI** for a modern, fast backend.
* Chose **Streamlit** for rapid Python-based frontend development.
